🚀 Tech Pulse 2026 – Event Registration Portal

Tech Pulse 2026 is a modern, responsive event registration web application built for the Department of Information Technology.

The platform allows students to register seamlessly while automatically storing responses in Google Sheets and providing instant WhatsApp group onboarding.

✨ Features

🎯 Clean and animated futuristic UI

📝 Real-time event registration form

📊 Google Sheets integration via Apps Script

✅ Success popup modal with student name

💬 Automatic WhatsApp group invitation

🔗 Instagram redirection integration

🌐 Fully responsive design

⚡ Deployed on Vercel

🛠 Tech Stack

HTML5

CSS3 (Custom + Tailwind)

JavaScript (Vanilla)

Google Apps Script

Google Sheets

Vercel Deployment
